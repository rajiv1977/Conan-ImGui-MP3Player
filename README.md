# Conan-ImGui-MP3Player
MP3 development based on ImGui and Microsoft libraries. 

# Prerequisite (Windows):
* Conan
* CMake (latest)
* Visual Studio 2022 (Win)

# Build (Window):
* Run **run_debug.bat** or **run_release.bat**
